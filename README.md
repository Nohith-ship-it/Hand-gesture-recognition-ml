# Hand-gesture-recognition-ml
A machine learning and computer vision project for recognizing hand gestures from images and real-time video using SVM and MediaPipe.

# ✋ Hand Gesture Recognition System

## 📌 Project Overview

This project develops a hand gesture recognition system that identifies different hand gestures from images or real-time video using machine learning and computer vision techniques.

---

## 🎯 Objective

To enable gesture-based human-computer interaction by recognizing hand gestures such as:

* Thumbs Up
* Peace
* Palm
* Fist

---

## 📂 Dataset

Hand Gesture Dataset (Kaggle)

Organized into gesture categories:

* thumbs_up
* peace
* palm
* fist

---

## ⚙️ Technologies Used

* Python
* OpenCV
* NumPy
* Scikit-learn
* MediaPipe

---

## 🔄 Workflow

1. Load and preprocess images
2. Extract features
3. Train classification model (SVM)
4. Evaluate accuracy
5. Real-time gesture detection using webcam

---

## 🧠 Models Used

* Support Vector Machine (SVM)
* MediaPipe Hand Tracking

---

## ▶️ How to Run

```bash
pip install opencv-python numpy scikit-learn mediapipe
python gesture_recognition.py
```

---

## 💻 Features

* Image-based gesture classification
* Real-time gesture recognition
* Simple and efficient implementation

---

## 🚀 Future Enhancements

* Deep learning (CNN) for higher accuracy

## ⚠️ Limitations
